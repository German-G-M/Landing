Requerimientos
Imaginemos que este proyecto corresponde a un repositorio desarrollado por el profesor y
ampliamente utilizado por la comunidad de desarrolladores. Los alumnos -al ver el proyectoconsideran
que hay varios cambios que podrían mejorar el contenido del repositorio.
Deciden colaborar con el proyecto, para esto:

1. Crear un fork del repositorio.
2. https://github.com/DireccionAcademicaADL/Landing .
3. Clonar el repositorio a nuestro computador.
4. Generar un cambios simples en el proyecto.
5. Commitear los cambios.
6. Hacer push a nuestro repositorio forkeado.
7. Crear un pull request describiendo los cambios implementados.
El docente analizará los pull requests y decidirá qué cambios implementar.
